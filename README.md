# Cardápio Digital - Zecas Pastelaria

## Como usar

- Abra o `index.html`.
- Para informar a mesa via QR Code, use o parâmetro `mesa` na URL:

`index.html?mesa=12`

## Como trocar o número do WhatsApp

- Edite `app.js` e altere a constante `WHATSAPP_NUMBER`.

## Como editar itens e preços

- Edite `menu-data.js`.

## Publicação / QR Code

- Publique esta pasta em qualquer hospedagem estática.
- Gere um QR Code apontando para a URL publicada (incluindo `?mesa=XX` se desejar).
